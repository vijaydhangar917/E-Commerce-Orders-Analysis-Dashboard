# E-Commerce-Orders-Analysis-Dashboard

## Project Overview
This project analyzes the sales performance and orders of a mobile accessories E-Commerce company using Power BI. The dashboard provides insights into sales trends, sales targets, order sources, and country-wise performance.

## Tools Used
- Power BI
- Power Query
- DAX

  ## Dataset Information
The dataset contains three tables:
1. Orders
2. Customers
3. Sales Targets

## Key Analysis Performed
- Sales trend analysis across months
- Country-wise sales performance
- Sales target achievement analysis
- Order source analysis
- Customer category insights
- Sales Manager performance analysis

## Dashboard Features
- Interactive slicers
- KPI cards
- Drill-through reports
- Map visualizations
- Monthly sales trends
- Target achievement charts

## Key Insights
- November recorded the highest sales and orders.
- Website was the most used order source.
- Noah Meier was the top-performing Sales Manager.
- Category D generated the highest sales.
- Category E had the youngest customers on average
